1. Copy the HrmSystem folder inside htdocs then paste it ex: C:\xampp\htdocs\HrmSystem 
2. Create a Database name "apsystem" in MYSQL 
3. Click the Database appsystem then click IMPORT THEN CLICK File to import the find C:\xampp\htdocs\HrmSystem\database\apsystem.sql then click Go
4. Be sure that the APACHE AND MYSQL IS RUNNING. If not then you need to click the xampp-control.exe and Click start APACHE, start MySQL
5. Go. to the browser and type http://localhost/HrmSystem/admin/index.php then Login with the username and password
	username = admin
	password = password
